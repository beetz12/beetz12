## My Approach to AI

I believe we're living through a **fundamental paradigm shift**. With solid CS fundamentals and AI as a collaborator, developers can pick up new stacks and ship production-ready apps in 2-3 weeks instead of months.

Here's the thing — I don't blindly trust AI. I **collaborate** with it. I review every line of code, I push back when it suggests bad patterns, and together we build solid architecture. It's like pair programming with the most knowledgeable but occasionally overconfident junior dev you've ever met.

**My tool evolution:** VS Code → Cursor → Windsurf → Claude Code → Now: **Claude Code + Google AntiGravity**

I'm a tinkerer at heart. I love experimenting with the latest technologies, tackling hard problems, and building tools that save people time. With AI, I now have the capability of an entire small development team — and I'm just getting started.
